# flutmax-validate

Validation for .maxpat files: JSON structure, static analysis, Node for Max runtime.

Part of the [flutmax](https://github.com/nordsound/flutmax) workspace.

## Overview

Provides multi-level validation for `.maxpat` files:

- **JSON structure** -- validates the `.maxpat` JSON schema
- **Static analysis** -- checks object names, connection validity, and type compatibility
- **Node for Max runtime** -- optional runtime validation by loading patches in Max via Node for Max
