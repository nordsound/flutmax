# flutmax-lsp

Language Server Protocol implementation for flutmax.

Part of the [flutmax](https://github.com/nordsound/flutmax) workspace.

## Overview

An LSP server for `.flutmax` files, providing IDE features such as diagnostics, completions, and hover information. Built on `tower-lsp` and designed for use with the [flutmax VS Code extension](https://github.com/nordsound/flutmax/tree/main/editors/vscode).

## Running

```bash
# The LSP binary communicates over stdio
flutmax-lsp
```
